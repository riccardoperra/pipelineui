# Supported GitHub Workflow features

> [!NOTE]
>
> The table below is an hand-made list of all features that PipelineUI should support according to the [GitHub workflow syntax](https://docs.github.com/en/actions/writing-workflows/workflow-syntax-for-github-actions) schema.
>
> If you encounter any problem, please fill an [issue](https://github.com/riccardoperra/pipelineui) on GitHub.
